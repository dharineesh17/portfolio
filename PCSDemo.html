Photo Capture Web App

This demo website showcases the core workflow of a custom-built Photo Capture Web App originally developed for documenting voter participation during an election day. The live app was used in offline environment (localhost server), and successfully handled over 1000+ photo captures in a single day at an average of 34.79 seconds per capture , ensuring high accuracy and throughput.

HTML5 CSS3 Tailwind-style UI JavaScript (WebRTC) PHP CSV / JSON XAMPP

Key Features

Fetch Details from Data: Retrieves participant details before capture.
Webcam Access with Live Preview: Uses WebRTC for real-time camera feed.
Smart Auto-Naming: Saves photos as serialnumber_name.jpg for easy organization.
Duplicate Capture Warning: Alerts if an entry is already captured.
Timestamp Logging: Stores capture time along with metadata in CSV and JSON files.
Live Progress Tracking: Analytics page shows percentage completed from total expected entries.
